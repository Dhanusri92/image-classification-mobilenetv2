# Image Classification using MobileNetV2 (CIFAR-10)

## Overview
This project is a Deep Learning image classification model built using PyTorch and Transfer Learning (MobileNetV2). It classifies CIFAR-10 images into 10 categories.

## Features
- Pretrained MobileNetV2 (Transfer Learning)
- Data Augmentation
- Training on subset dataset
- Evaluation metrics (Accuracy, Precision, Recall, F1-score)
- Training curves visualization
- Model saving (.pth file)

## Tech Stack
- Python
- PyTorch
- Torchvision
- Scikit-learn
- Matplotlib

## Dataset
- CIFAR-10 dataset (50,000 train images, 10,000 test images)

## Results
- Trained on 5,000 images
- Tested on 1,000 images
- Achieved good accuracy using transfer learning

## Files
- Cats_vs_Dogs_Image_Classification.ipynb
- mobilenetv2_cifar10.pth
